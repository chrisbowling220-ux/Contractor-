- **iOS App Store is NOT set up** — Apple doesn't accept PWA/TWA-style wrappers and
  won't list a PWA directly. iOS would need a separate native shell (e.g. Capacitor).
  For now, iPhone users can "Add to Home Screen" in Safari from builderspro.cc.
- Trademark note: avoid naming retailers (Home Depot/Lowe's) in store metadata —
